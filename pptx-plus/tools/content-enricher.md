# 内容丰富器

按指定深度等级丰富分享大纲的详细内容。

## 使用场景

- 大纲已确认，丰富级别已选定
- PPT 生成前，为幻灯片填充有意义的内容

## 输入

- 已确认的分享大纲（Markdown 格式）
- 丰富级别：L1、L2、L3 或 L4
- 原始用户材料（所有来源）

## 丰富级别

### L1 精简（Concise）
- **内容**：每章 1-2 句，每小节 1-2 个要点
- **来源**：仅 AI 常识
- **适用**：用户想要快速草稿，自己填充细节
- **输出**：轻量级大纲含简要描述

### L2 标准（Standard）
- **内容**：每小节 3-5 个要点，含背景和意义
- **来源**：AI 常识 + 用户提供的材料
- **适用**：通用商务演示，平衡深度
- **输出**：完整的演示级内容

### L3 丰富（Detailed）
- **内容**：含数据、案例、建议的详细展开
- **来源**：AI 知识 + 用户材料 + 网络搜索
- **适用**：团队分享、重要演示
- **输出**：含支撑证据的全面内容

### L4 定制（Customized）
- **内容**：行业数据、竞品对比、最佳实践
- **来源**：AI 知识 + 用户材料 + 深度网络搜索
- **适用**：战略规划、竞品分析、高管简报
- **输出**：含市场情报的战略级内容

## 工作流程

1. 解析大纲结构（章节和小节）
2. 按选定级别丰富每个小节：
   - L1：总结为 1-2 句
   - L2：扩展为要点 + AI 知识 + 用户材料
   - L3：扩展为详细数据 + 搜索统计
   - L4：深度研究 + 竞品分析 + 行业数据
3. L3/L4：使用网络搜索查找相关数据、案例和基准
4. 标注内容来源：
   - `[用户提供]` — 用户原始材料
   - `[AI知识]` — AI 知识补充
   - `[网络搜索]` — 网络搜索结果
5. 以完整 Markdown 文档格式输出

## 质量标准

- 每个丰富后的章节应达到演示可用级别
- 同一级别内各小节保持一致的深度
- 为团队受众优先提供可行动的洞见
- 仅输出有把握的准确内容，绝不凭空捏造数据或不确定的表述
