# 开发流程

```
启动开发Agent（任务：编写实现代码） ← 阶段五：开发流程开始
    │                     ▲
    ├──发现架构设计问题      │
    │       │             │
    │       ▼             │
    │   启动架构Agent修复   │
    │       │             │
    │       └─────────────┘
    ▼
启动实现评审Agent（任务：代码实现评审）
    │
    ├──不通过 → 启动开发Agent修复（循环）
    │       │
    │       ├──循环5次内 → 继续循环
    │       │
    │       └──5次后仍不通过 → 主控决断
    │
    └──通过
        │
        ▼
    进入阶段六：测试流程
```
