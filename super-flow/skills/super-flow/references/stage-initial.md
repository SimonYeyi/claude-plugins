# 初始流程

```
用户需求 ← 初始阶段：初始流程开始
    │
    ▼
探索项目现状（读取项目结构 + docs/superflow/specs/ + 代码仓库）
    │
    ▼
判断与需求的重叠程度
    │
    ├── 完全重复 → 告知用户功能已存在，建议直接使用（流程结束）
    │
    ├── 部分重叠 → 标注重叠情况（互补/替代/增强） ──┐
    │                                               │
    └── 无重叠 ─────────────────────────────────────┘
    │
    ▼
进入阶段一
```
